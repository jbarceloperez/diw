Imágenes y multimedia

La etiqueta <img> inserta imágenes. Siempre usar alt para accesibilidad.

Atributos importantes: src (ruta), alt (texto alternativo), width, height, loading="lazy".

También existen <audio> y <video> para multimedia con controles nativos.

Buenas prácticas y accesibilidad

  • Usa etiquetas semánticas para estructurar (header, nav, main, footer).
  • Incluye atributos alt en imágenes y label asociados a inputs.
  • Evita usar tablas para maquetación; son para datos tabulares.
  • Prueba el teclado: los formularios y enlaces deben ser navegables sin ratón.
