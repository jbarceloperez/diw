2. Gradientes

Gradientes lineales, radiales, repetidos y patrones construidos con gradientes.

Linear gradient (lineal)

background: linear-gradient(90deg, #ff7a18, #af002d);

Radial gradient (radial)

background: radial-gradient(circle at center, #89f7fe, #66a6ff);

Repeating gradient (repetido)

repeating-linear-gradient(0deg, #eee 0 10px, #ddd 10px 20px);

5. Transformaciones (transform)

Transformaciones 2D y 3D: scale, translate, rotate, skew y matrix(). Se combinan con transform-origin.

Conclusión y notas

  • Acuerdate de cómo afecta la propiedad al flujo (por ejemplo, transform no afecta el flujo, mientras que position sí).
  • Usa transiciones para micro-interacciones y animaciones para movimientos más complejos o repetidos.
  • Combina efectos para lograr diseños atractivos y modernos.
