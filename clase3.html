3. Introducción a los estilos CSS

Existen tres formas de aplicar estilos CSS:

  • En línea: dentro del propio elemento HTML con el atributo style.
  • Internos: en la sección <style> del documento (como este ejemplo).
  • Externos: enlazando un archivo .css con <link>.

Ejemplo de enlace a hoja de estilo externa

Para enlazar un archivo externo, usaríamos:

  <link rel="stylesheet" href="estilos.css">

En ese archivo estilos.css podríamos definir reglas como:

  body {
    background-color: #e0f7fa;
  }
  h1 {
    color: darkblue;
  }
